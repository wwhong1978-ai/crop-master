# 图片裁剪工具 V6.1 - 使用说明

## 📌 快速开始

1. 安装依赖：`pip install -r requirements.txt`
2. 运行程序：`python main.py`
3. 点击 **"打开文件夹"** 选择图片所在目录
4. 设置裁剪尺寸和输出目录，开始裁剪

## 🔧 功能说明

### 1. 打开图片文件夹
- 点击左侧 **"打开文件夹"** 按钮
- 支持格式：`jpg`、`jpeg`、`png`、`bmp`（大小写均可）
- 自动加载缩略图列表，默认选中第一张

### 2. 调整裁剪选框
| 操作 | 效果 |
|------|------|
| **鼠标拖拽** 选框内部 | 移动选框位置 |
| **鼠标滚轮** 向上/下 | 放大/缩小选框 |
| 右键点击图片区域 | 直接执行单图裁剪 |
| 点击左侧缩略图 | 切换查看该图片 |

### 3. 设置裁剪尺寸
- **裁剪宽度 / 裁剪高度**：设置输出图片的像素尺寸
- **选框大小**：设置选框占图片的百分比（10% ~ 100%）
- **应用尺寸到选框**：根据宽度/高度的比例调整选框形状
- **居中**：将选框移到图片正中心

### 4. 单图裁剪
1. 调整选框到合适位置和大小
2. 选择输出目录
3. 点击 **"单图裁剪"**（或右键图片）
4. 自动保存到输出目录，并跳转到下一张

### 5. 批量裁剪
1. 设置好裁剪尺寸和选框百分比
2. 选择输出目录
3. 点击 **"批量裁剪"**
4. 自动居中裁剪所有图片，进度条显示进度

---

## 💡 使用技巧

- **选框百分比 = 100%**：保留图片最大可用区域（居中裁剪）
- **选框百分比 = 90%**：默认值，去掉边缘 10%，适合去除水印/边框
- **选中状态**：已处理的缩略图背景会变灰，方便识别
- **配置自动保存**：裁剪尺寸、输出目录、上次文件夹等设置关闭时自动保存
- **配置文件位置**：`%USERPROFILE%\图片裁剪工具_配置.json`

## ⚠️ 注意事项

1. **必须先选择输出目录**，否则裁剪按钮不可用
2. 输出文件名格式：`原文件名_时间戳.jpg`，不会覆盖原图
3. 输出格式固定为 **JPEG（质量 90）**
4. 批量裁剪按**选框百分比居中裁剪**，不使用手动调整的选框位置
5. 图片数量较多时，缩略图采用**懒加载**，滚动到才会加载
